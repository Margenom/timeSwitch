Система планирования с ограничением по времени
Любое дело отвликает внимание и с короткими делами надо разбираться сразу (нападай на слабых врегов уменьщая их количество)
Как устроена, есть 2 файла
	- Надо или можно сделать, похож на cron: минута[0-59], час[0-23], день[1-31], месяц[1-12], день недели[0-7], длительнось[минуты], описание[просто текст, но +контекст %ключ:значение !категория #приоритет[цифра]]
	- Сделано, обычный фаил в DSV(разделитель \t), но перый символ определяет что это за запись и к какой категории она относиться
		Нередактируеться, просто лог
		- начало выполнения планового задания, всё кроме utime берется из bisy файла тк возможны его дальнейшие правки 
			<, от[utime], длительность(планируеться, как в файле 1)[секунды], описание[такоеже как в первом файле]
		- неплановые действия может быть несколько
			\t, от[utime], длительность[секунды], описание[такоеже как в первом файле]
			Для некоторого упрашения системы и улучшения использование пользователем возможна частичная запись как:
			\t, от,
			Чтобы потом можно было дописать время выполниения непланового действия как {now-$от}
		- конец планового задания
			>, от(посути фактическое время завершения)[utime], реальная длительность задания(если задание растягивалось)[секунды], коментарий[просто текст]
Редактируеться 1 файл как обычный текстовый через обычный текстовый редактор

Команды:
- Планируемо
		Показать список задач для выполнения
		Приоритет(По времени, По приоритету, Остальные), Длительность в мин, Название, Параметры :Форма
	list [<regexp pathern>] [-n numbers, def on] [-c=<category>, def all] [-<key>=<value>] [-p=<priority>] [-x=<context>]
		Выбрать задачу что будет выполняться сейчас
	select <номер из list> 
		Закончить выполнение задания 
	end [<comment>]
		Запустить таймер по окончанию задачи (запускает команду, вфоне и после её завершения выводит сообщение мол завершить выполнение или продлить, тогда завершать только end)
		поумолчанию звуковой сигнал
	timer [<commmand>]
- Непланируемо
		Сообщить что ты чтото начал делать не по плану
	stun
		Отчитаться за совершенные действия
	tell	
		Запустить сщетчик перерыва (программа ждет прирывания)
		возможно напаминапие раз в определенные промежутки времени о том что сейчас пауза.
	wait [<описание что буду делать>] [-i=<interval, micro alarm>]
- Общие системные
		Показать выполняемые задачи сейчас, начало, осталось времени (или сверх) и список неплановых дел
	now
		Выводит статистику за время t по параметру s
	stat [-t=<time>] [-s=<stat tipe, by cat, by key, by priority, by context, by time(def)>] [-c=<category>, def all] [-<key>=<value>] [-p=<priority>] [-x=<context>]
		Проверка всё ли влазит в день
	check
